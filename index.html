¿Que es Bootstrap?

Bootstrap es un framework de CSS, este es utilizado en el area de desarrollo FrontEnd, va en conjunto de JS y CSS para darle una apariencia y interactividad mas creatica al estructurado HTML. y crear sitios responsive.

Elementos

Barra de Navegación

Es uno de los mas comunes elementos que podemos encontrar en una pagina web ya que nos brinda una barra comunmente en la parte superior del sitio el cual nos sirve para poder navegar entre los diferentes apartados que posee la pagina.

Menús Desplegables

Este elemento añade una característica extra al elemento anterior ya que a alguno de los recuadros de la barra de navegación se le agrega otro tipo de sub menú de manera desplegable.

Paginación

Nos permite tener en botones la cantidad de páginas que dispone el sitio para asi poder movernos entre ellas de manera mas facil e intuitiva.

Contenedores

Los contenedores son un bloque de construcción fundamental de Bootstrap que contienen, rellenan y alinean su contenido dentro de un dispositivo o ventana gráfica determinada.

Ejemplos

Comience rápidamente un proyecto con cualquiera de nuestros ejemplos que van desde el uso de partes del marco hasta componentes y diseños personalizados.

Temas

Cada tema está diseñado como una extensión de Bootstrap, creado para un conjunto específico de problemas. Esto significa no solo ampliar los componentes básicos de Bootstrap, sino también agregar componentes, utilidades y complementos completamente nuevos.

Iconos

Biblioteca de iconos de código abierto, de alta calidad y gratuita con más de 1800 iconos. Inclúyalos como desee: SVG, sprite SVG o fuentes web. Úsalos con o sin Bootstrap en cualquier proyecto.
